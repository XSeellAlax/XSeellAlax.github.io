简单工厂模式

为什么要学设计模式

C++ 中的设计模式可帮助开发人员创建可维护、灵活且易于理解的代码。它们浓缩了经验丰富的软件架构师和开发人员的专业知识和经验，使新程序员更容易遵循既定的最佳实践。

简单工厂模式实例

在本教程中，您将学习如何使用 C++ 实现简单工厂模式。简单工厂模式是一种创建对象的模式，它提供了一个静态工厂方法，该方法返回指定类的实例。
步骤：

  1. 创建一个接口。
  2. 创建一个类，该类实现接口。
  3. 创建一个类，该类包含一个静态工厂方法，该方法返回接口的实现类的实例。
  4. 创建一个类，该类包含一个构造函数，该构造函数初始化接口的实现类的实例。
